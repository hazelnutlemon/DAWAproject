담당 : 조예슬(조장), 김성종(조원)

Flask를 사용하여 웹 어플리케이션 구현
각 페이지 간 연동 문제로 조장 단독 구현 및 조원 서포트로 진행.

분석 그래프의 경우, matplotlib 패키지를 이용해 이미지 출력.
기준 평균 선은 전체 작품에 대한 해당 월 평균 점수 그래프.
막대 그래프는 해당 작품에 대한 감성분석 점수.

[ 구성 ]
1. Main : 메인 홈 화면 - 검색, 이달의 작품 Top10, 메뉴 아이콘
2. Menu : 메뉴 화면 - 연재처별 작품 목록, 로맨스/판타지 장르별 작품 목록으로 이동할 수 있는 아이콘
3. 연재처 List : 연재처별 작품 목록 화면 - 제목 순 나열로 작품의 간단한 정보(표지, 제목, 작가, 장르)
4. 장르 List : 장르별 작품 목록 화면 - 감성분석 점수를 기반으로 내림차순으로 나열. 제공 정보는 연재처 List와 동일.
5. 작품 List : 전체 작품 목록 화면 - 삽입 순으로 나열.
6. Content : 작품 정보 화면 - 작품의 기본 정보와 분석 결과를 제공.
